# BigBasket Clone

A fully functional and responsive **BigBasket Clone** built with **HTML, CSS, JavaScript, and Bootstrap**.

### Features

- User Login with Email + OTP Verification
- Browse products by categories
- Add items to Cart
- Increase/Decrease quantity in Cart
- Real-time price calculation
- Checkout functionality with success message
- Responsive design (Mobile + Desktop friendly)
- Clean and modern UI similar to BigBasket

### Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Responsive Design

### How It Works

1. User enters email and receives OTP for login
2. Browse products and add to cart
3. Adjust quantity in cart (price updates automatically)
4. Click **Checkout** → Order placed successfully + Cart cleared
